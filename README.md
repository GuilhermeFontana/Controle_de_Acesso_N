# Controle_de_Acesso
 v0.0 - CANCELADO
 
 
# Descrição
----

```
Este seria um projeto de Controle de Acesso de Pessoas, para um edifício comercial. Ele teria integração com
5 leitores biométricos e 2 catracas.

Ao final do projeto, ele seria uma hibridização destes dois projetos (Integracao-com-Equipamentos(backend) e 
Telas-Finais(frontend)).
```

# Observações
```
Somente foram programadas as transições de telas, sem nenhuma programação efetiva.
 
Entretanto quase toda API de comunicação com os equipamentos esta concluída e funcional (em torno de 75%/85%),
mas não conectada ao frontend.

Outro ponto também é que o software já se comunica com o banco de dados, mas realiza apenas a conexão, sem a 
possibilidade de realizar qualquer consulta, alteração a inserção de dados no mesmo

```
